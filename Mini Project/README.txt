# Employee Record Management System

This is a simple Employee Record Management System implemented in C. The system allows you to perform basic operations on employee records, including adding, deleting, displaying, and modifying records. It provides a command-line interface for user interaction.

## Features

- **Add Record**: You can add employee records by entering the employee's name, age, salary, and ID. Records are stored in a binary file.

- **Delete Record**: You can delete employee records by specifying the name of the employee you want to delete. The system will remove the specified record from the file.

- **Display Records**: This option allows you to view all the employee records stored in the file. It displays the employee's name, age, salary, and ID.

- **Modify Record**: You can update the information for a specific employee by providing their name. You can change their name, age, salary, and ID.

- **Exit**: Exit the program.

## Getting Started

1. Compile the code using a C compiler. For Windows, you can use a C compiler like Dev-C++, Code::Blocks, or Visual Studio.

2. Run the compiled executable.

3. Use the menu to navigate and perform operations on employee records.

## Usage

- Follow the on-screen menu to choose the desired operation.
- When adding or modifying a record, enter the employee's details as prompted.
- When deleting a record, provide the name of the employee you want to delete.
- To exit the program, choose the "Exit" option from the menu.

## File Management

- Employee records are stored in a binary file named "data.txt."
